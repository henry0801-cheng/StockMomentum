# 資金管理
- **總資金**：2千萬元（固定）
- **最大持倉檔數**：S_H檔，限制同時持有的最大股票數量
- **進場S_B**：總資金 / S_H檔

# 交易成本
- **交易稅**：0.1%
- **滑價**：0.3%

# 回測優化 (Ant Colony Optimization)
- **演算法**：螞蟻演算法 (ACO)
- **目標函數**：最大化 (CAGR,Calmar Ratio)組合
- **Python 終端機輸出**：每一個螞蟻演算法世代的最佳年化報酬、Calmar ratio 及最佳參數
- **特殊參數**：加入 ALPHA (費洛蒙重要性因子)，以控制探索與利用的平衡

# 輸入/輸出
## 輸入資料
- Excel 檔案：
  - **P 工作表**：還原收盤價 (Adjusted Close)
  - **Q 工作表**：成交量 (Volume，單位：張)

## 輸出結果
- Excel 檔案：檔名設定為策略名稱
  - **Trades**：詳細交易明細 (買、賣日期/價格/股數、損益、報酬率、買進參數詳細內容(可多欄)、原因)
  - **Equity_Curve**：每日權益變化數據，並用 Python 程式庫畫出權益曲線圖 (含回撤陰影)
  - **Equity_Hold**：每日持股檔數與明細
  - **Daily_Record**：進場與持倉管理的每日檢查紀錄
  - **Daily_Candidate**：每日符合條件的股票及檔數
  - **Summary**：策略績效總結 (CAGR, MaxDD, Calmar, Win Rate, 最佳參數等)


# 程式架構
- 單一 Python 檔案：檔名設定為策略名稱
- 包含：
  - **資料讀取**
  - **策略邏輯類別 (Strategy)**
  - **優化演算法類別 (AntColonyOptimizer)**
  - **主程式**
- 程式碼包含詳細繁體中文註解
- 結構嚴謹簡潔、易讀易改，避免 FutureWarning（未來版本相容性警告）
- 程式內必須包含旗標切換ACO與手動在程式內輸入參數，讓程式依據不同參數方法產出結果。
- **參數位置**：策略手動、ACO參數與螞蟻演算法參數在程式碼中的位置相近，方便修改
